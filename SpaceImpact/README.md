# Space Impact

## Overview
Space Impact is a 2D space shooter game where players control a spaceship, battle against enemies, and collect power-ups. The game features various levels, a final boss, and a scoring system.

## Features
- **Player Control**: Navigate the spaceship using keyboard inputs.
- **Enemies**: Various enemy types with different behaviors.
- **Laser Shooting**: Fire lasers to defeat enemies.
- **Explosions**: Visual effects for explosions upon enemy defeat.
- **Pickups**: Collect health and shield pickups to enhance gameplay.
- **Final Boss**: A challenging final boss encounter.
- **Parallax Background**: Enhanced visual experience with layered backgrounds.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd SpaceImpact
   ```
3. Open the solution file in your preferred IDE.
4. Restore the project dependencies and build the project.
5. Run the game to start playing!

## Usage
- Use the W, A, S, D keys to move the spaceship.
- Press the Spacebar to shoot lasers.
- Collect pickups to improve your health and gain shields.
- Defeat enemies and the final boss to win the game.
